## Installation :
if you are on x86 :
```bash
make
make all
make run
```
if you are on arm cpu :
```bash
make
make iso
make run32
```
if u want clean every compiled file of the project do :
```bash
make clean
```

dont forget to look at the requirement in /docs to install every package
